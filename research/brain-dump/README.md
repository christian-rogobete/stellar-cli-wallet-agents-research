# Historical brain-dump — not an RFP specification

This folder contains the operator's original, informal brain-dump of initial ideas for the wallet, written early in the research process (2026-04-16). It predates the formal analysis in `analysis/03-requirements.md` and is **NOT** a statement of what the RFP must require — the SDF RFP lead owns the requirements.

Each bullet was subsequently seed-mined into atomic, tagged, classified requirements. For the disposition of each bullet (converted, split, merged, rejected, or deferred), see `analysis/03-requirements.md` Appendix A and `analysis/03-requirements-addendum.md`.

Preserved here for research-record completeness. Citations elsewhere in the repository reference bullets by line number (for example, `research/brain-dump/requirements.md:3` points at bullet 1, "self custody wallet controlled by the ai agent").
